BLUEPRINT PENGEMBANGAN APLIKASI RUNITY (MODERN RUNNING APP)
==========================================================

1. INFORMASI PROYEK
-------------------
Nama Aplikasi   : Runity
Platform        : Flutter (Android, iOS, & Web Support)
Target Pengguna : Pelari kasual hingga profesional (Masyarakat Modern)
Fokus Utama     : Akurasi Data, Pengalaman Visual Premium, dan Personalisasi

2. ARSITEKTUR TEKNIS (TECH STACK)
---------------------------------
- Framework: Flutter (Dart)
- State Management: Riverpod (Reactive state handling)
- Database: 
    * Local: Hive (NoSQL) - Sangat cepat untuk menyimpan koordinat GPS.
- Map Engine: Flutter Map (OpenStreetMap) - Fleksibel & Bebas API Key.
- Lokalisasi: Custom AppTranslations (Support Bahasa Indonesia & English).
- Animasi: flutter_animate (Framer-motion style effects).
- Social: share_plus (Integrasi berbagi ke media sosial).

3. MODUL FITUR TERIMPLEMENTASI (LATEST)
---------------------------------------
- SMART DASHBOARD:
    * Dynamic Mesh Gradient Background dengan Moving Radial Glow.
    * Live Run Card: Memantau lari aktif langsung dari halaman utama.
    * Last Run Reference: Akses cepat ke detail aktivitas terakhir.
    
- ADVANCED TRACKING SYSTEM:
    * Dynamic UI Morphing: Panel statistik berubah menjadi bar horizontal (lanskap) saat peta diperbesar.
    * Real-time Metrics: Jarak, Waktu, Pace, dan Kalori.
    * Full Map Mode: Visualisasi rute penuh tanpa gangguan panel besar.

- PERSONALIZED PROFILE:
    * Custom Bio: Pengguna dapat menambahkan biodata diri.
    * Photo Integration: Ganti foto profil (Web & Mobile compatible).
    * Security: Logout dengan dialog konfirmasi & reset data lokal otomatis.

- ACTIVITY HISTORY:
    * Detail View: Lihat rute lari lama dengan fitur Toggle Panel statistik.
    * Social Share: Bagikan ringkasan lari ke Instagram/WhatsApp/Twitter.

5. RENCANA FITUR SOSIAL (FUTURE)
--------------------------------
- PROFILE STORIES: Fitur berbagi momen lari singkat (foto/video 24 jam) ala Instagram.
- ATHLETE CHAT: Sistem pesan instan terintegrasi untuk diskusi rute atau janjian lari ala WhatsApp.
- SOCIAL NETWORK: Sistem pertemanan (Following/Followers) untuk membangun komunitas atlet ala Instagram.
